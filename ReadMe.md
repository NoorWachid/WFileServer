# WFileServer

Complete rewrite simplified version of [HFS](https://www.rejetto.com/hfs/).

## Installation
*IT REQUIRES NODE.JS INSTALLED ON YOUR SYSTEM*

### Gnu/Linux
- `git clone https://github.com/NoorWachid/WFileServer`
- `cd WFileServer`
- `sudo ln -s $PWD/wfs /usr/bin`
